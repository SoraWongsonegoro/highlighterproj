How to use

To start, go to any webpage and select some text by left-clicking and dragging. Floating above the blue-selected text, a “save” icon should appear. Click the save icon to “highlight” the quote and save it to your dashboard.

After saving a quote, you can click on the quote in the popup to view more details, including the full text, source, and date saved. From there, you can also copy the quote or save it as a PNG image.

To organize your quotes, you can move them to a collection. First, navigate to the Collections tab. Then, create collections based on how you wish to organize your quotes. Next, return to the “All Quotes” tab, find the quote you wish to sort, hover over the quote, and click the folder button. A selection screen will allow you to choose which collection the quote goes into.

There are also options to copy and delete the quote when a quote is hovered over. In the collections tab, you can see all the quotes in one collection by clicking on a collection tab. Additionally, you can also delete a collection without deleting the quotes inside it.


To find quotes from a specific website, you can see quotes sorted by the website they are from in the “Quotes by Page” tab. To find quotes by a specific keyword, there is a search bar function that will only show quotes with that specific keyword or phrase.


For performing the same operation on multiple quotes, there is a select feature that allows you to batch select quotes to move, copy, export, or delete them. To access the selection view, click the top right “Select” button, which will make checkboxes appear next to quotes. Click on the checkbox to select the quote. There is also a select all button at the bottom of the screen. To exit the selection view, click the “Select” button again.

For faster highlights, use Ctrl+Shift+S to highlight and save selected text. In the settings, you can customize quote sorting and the color of highlights.


Finally, you can export your quotes as a database file that can imported into other applications. Similarly, you can import database files from other users to share quotes.
